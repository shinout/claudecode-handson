1 Claudeって何？

まずはAIとClaude Codeについてざっくり理解しよう。

AIチャットは使ったことある？

Geminiで「〇〇を要約して」「メールの文面考えて」とやったこと、ありますよね。
あれはブラウザ上でAIと会話するツールです。

ClaudeはAnthropicという会社が作ったAIで、Gemini（Google）やChatGPT（OpenAI）と同じ種類のものです。

たとえるなら：
Gemini = Googleが作ったAIアシスタント
ChatGPT = OpenAIが作ったAIアシスタント
Claude = Anthropicが作ったAIアシスタント
どれも「賢いAI」だけど、作った会社が違う。スマホでいうiPhoneとPixelとGalaxyの関係。

じゃあ Claude Code って？

普通のClaude（claude.ai）はブラウザで会話するだけ。でも Claude Code はちょっと違います。

Claude Codeはあなたのパソコンの中で動くAIです。ファイルを作ったり、編集したり、コマンドを実行したり、実際に「手を動かす」ことができます。

たとえるなら：
普通のClaude = 隣にいるアドバイザー。「こうしたらいいよ」と教えてくれる
Claude Code = 隣にいるプログラマー。「やっとくね」と実際にやってくれる
💡
Claude Codeは「ターミナル」（黒い画面）の中で動きます。ターミナルって何？というのは次のチャプターで説明します。

何ができるの？

たとえばこんなことを頼めます：

  • 「Webページを作って」 → HTMLファイルを作ってくれる
  • 「このファイルの内容を説明して」 → コードを読んで解説してくれる
  • 「エラーが出た、直して」 → エラーを読んで原因を見つけて修正してくれる
  • 「毎週の集計を自動化するスクリプト作って」 → Pythonスクリプトを作ってくれる

プログラミングの知識がなくても、日本語で頼むだけでOKです。

2 インストール

パソコンにClaude Codeを入れよう。15分くらいで終わります。

ターミナルを開こう

「ターミナル」は、パソコンに文字で命令を出すための画面です。

たとえるなら：
普段のパソコン操作 = マウスでアイコンをクリック（GUI）
ターミナル = キーボードで文字を打って命令（CUI）
どちらも同じことができるけど、プログラミングの世界では文字のほうが速い。

Windowsでは PowerShell という名前のターミナルを使います。

1
キーボードの Win キーを押す（左下のWindowsマーク）
2
「powershell」と入力する
3
「Windows PowerShell」が出てくるのでクリックして開く
✅
青い背景の画面が出たら成功！ここに文字を打って Enter を押すと、コマンド（命令）を実行できます。

Macでは「ターミナル」というアプリを使います。

1
キーボードの Command + Space を押す（Spotlight検索が開きます）
2
「terminal」と入力する
3
「ターミナル」が出てくるので Enter で開く
✅
白い（または黒い）画面が出たら成功！ここに文字を打って Enter を押すと、コマンド（命令）を実行できます。

バージョン番号が表示されたら成功です！

4 Git & GitHub

作ったものを保存・公開する方法を学ぼう。

なんで Git / GitHub が必要なの？

さっき作った自己紹介ページ、今はあなたのパソコンの中にしかありません。

  • パソコンが壊れたら消えてしまう
  • 他の人に見せられない
  • 「さっきのバージョンに戻したい」ができない

これを解決するのが Git と GitHub です。

たとえるなら：
Git = Googleドキュメントの「変更履歴」機能。いつ、何を変えたかを記録してくれる
GitHub = Google ドライブ。ファイルをクラウドに保存して、他の人と共有できる

つまり Git + GitHub = 「変更履歴付きのクラウド保存」

Git をインストール

1
以下のリンクからGit for Windowsをダウンロード
https://gitforwindows.org/
2
ダウンロードしたファイルを開いて、「Next」を連打でインストール
3
PowerShellを一度閉じて開き直してから、確認：

ターミナルで以下を実行してください。Macの開発ツール一式がインストールされます：

xcode-select --install

ポップアップが出たら「インストール」をクリック。数分かかります。
完了したら確認：

git --version

git version 2.xx.x と出ればOK！

GitHub CLI（gh）をインストール

GitHub CLI は、ターミナルからGitHubを操作するためのツールです。
リポジトリの作成やログインがコマンド1つでできるようになります。

1
PowerShellで以下を実行してインストール：
winget install --id GitHub.cli
💡
winget はWindows標準のアプリインストーラーです。「アプリストア」のコマンド版だと思ってください。
2
PowerShellを一度閉じて開き直してから、確認：

まず Homebrew（Mac用のアプリインストーラー）を入れます：

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
💡
Homebrew はMac定番のアプリインストーラーです。「アプリストア」のコマンド版だと思ってください。パスワードを聞かれたら、Macのログインパスワードを入力してください。
⚠️
インストール完了後、画面に「Next steps」として eval から始まるコマンドが表示されることがあります。その場合はそのコマンドもコピペして実行してください。

Homebrewが入ったら、GitHub CLIをインストール：

brew install gh

確認：

gh --version

gh version 2.xx.x と出ればOK！

GitHub アカウントを作ろう

https://github.com/signup にアクセス
2
メールアドレス、パスワード、ユーザー名を入力してアカウント作成

Git の初期設定

Gitに「あなたが誰か」を教えてあげます。名前とメールアドレスを自分のものに変えて実行してください：

git config --global user.name "あなたの名前"
git config --global user.email "your-email@example.com"
⚠️
メールアドレスは GitHubの登録に使ったもの と同じにしてください。

GitHub にコードをあげてみよう

ここからはClaude Codeに手伝ってもらいましょう！
まず cd ~/my-first-project で作業フォルダに移動してから、Claude Codeを起動：

cd ~/my-first-project
claude

そして、こう頼みます：

このフォルダをGitリポジトリにして、GitHubに「my-first-project」という名前のリポジトリを作って、コードをプッシュして。手順を1つずつ説明しながらやって。

Claude Codeが以下のことを1つずつ説明しながらやってくれます：

  1. git init — このフォルダをGitで管理し始める
  2. git add . — 変更したファイルを「記録対象」にする
  3. git commit — 変更を記録する（セーブポイントを作る）
  4. git push — GitHubにアップロードする
たとえるなら：
git init = 「このフォルダ、変更履歴つけるね」と宣言
git add = 「このファイルたちを保存対象にするね」
git commit = 「今の状態をセーブ！」（ゲームのセーブと同じ）
git push = 「セーブデータをクラウドにアップ」
💡
途中でGitHubの認証を求められることがあります。ブラウザが開いたら画面の指示に従ってログインしてください。

GitHub で確認しよう

プッシュが成功したら、ブラウザで GitHub を開いてみましょう：

https://github.com/あなたのユーザー名/my-first-project

さっき作った自己紹介ページのコードが表示されていたら大成功！
あなたのコードが世界に公開されました。
